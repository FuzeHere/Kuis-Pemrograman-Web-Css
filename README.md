## Pengantar singkat

Berikut ringkasan modifikasi yang dibuat pada proyek ini (komentar di `style.css` dan `index.html`):

- **Style (`style.css`)**: menyesuaikan font dan ukuran (`font-family: Arial`, `font-size: small`), komentar penanda perubahan pada beberapa aturan (`/* ccs vanila pertama/ kedua/ ketiga */`), mengatur warna header/footer (`#280872`), efek hover pada link navigasi (background biru), batas kanan pada item navigasi, `container` dengan `max-width: 900px`, gambar menu dibatasi `max-width: 40%`, dan footer diposisikan fixed di bawah.
- **Dashboard (`index.html`)**: menambahkan link CDN Bootstrap, menghubungkan `style.css`, menyisipkan komentar HTML untuk menandai perubahan (`<!-- css bootstrap pertama -->`, `<!-- css bootstrap kedua -->`), penggunaan gambar pada item menu, serta beberapa gaya inline seperti `style="width: 50em;"` pada card.

Jika Anda ingin, saya bisa memperbarui tautan menjadi URL GitHub Pages atau menambahkan petunjuk menjalankan lokal. Beri tahu format tautan online yang Anda inginkan.
